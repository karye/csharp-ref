# Monogame-projekt\*

## Skapa ett projekt

Skapa en ny mapp och öppna den. Kör följande kommando i terminalen (dubbelkolla att terminalen är i rätt mapp). Byt ut "MittProjektNamn" mot ett eget namn.

```
/dotnet new mgdesktopgl -o MittProjektNamn
```

## Projektstruktur\*

* Program.cs
* Game-klass
  * Initialize
  * LoadContent
  * Update
  * Draw
